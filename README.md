# vscode-yuque

vscode-yuque 是一个在 vscode 中编辑与展示“[雨雀](https://www.yuque.com/)”的插件（仅供内部使用）

## 使用

安装 vscode-yuque 后，vscode 将以雨雀文档的方式展示 .yuque 后缀的文件，并可在 vscode 中以雨雀的方式编辑该文档。
